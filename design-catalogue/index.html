Core Features

All designs implement the same core functionality with different visual approaches.

💻

Terminal-First

The terminal is THE interface. Talk to navigate, execute tasks, and manage your workspace through conversation.

☑

Task Management

Kanban-style task tracking with drag-and-drop. Click tasks to assign to agents.

💬

Multi-Agent Chat

Chat with Claude Code and Codex agents. View tool calls, approvals, and streaming responses.

⚲

Git Integration

View diffs, stage changes, switch branches, and commit from the interface.

🕑

Session History

Resume conversations with /resume. Track changes made by each agent session.

📄

File Browser

Browse workspace files. See what agents are reading and modifying in real-time.
